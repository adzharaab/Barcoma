My Learning Journey

June

In June, I learned about URLs and the World Wide Web. I understood how websites are connected online and how URLs help users navigate.

Challenge: remembering all the website structures was tricky.

July

In July, I started practicing basic HTML structure and creating simple web pages with headings, paragraphs, and lists.

Challenge: keeping my code organized and avoiding tag mistakes was hard at first.

August

In August, I explored HTML tags and links. I learned to create headings, paragraphs, and hyperlinks.

Challenge: organizing links properly and avoiding broken paths.

September

September focused on HTML tables. I learned to display data neatly in rows and columns.

Challenge: styling tables for readability.

October

In October, I learned about login boxes and input types. Creating forms helped me understand user interaction.

Challenge: validating inputs correctly.

November

November was about website planning. I learned to sketch layouts and plan content flow.

Challenge: designing a clear and consistent structure.

December

In December, I learned about multimedia. Adding images, videos, and audio made websites more dynamic.

Challenge: keeping media responsive and loading fast.

January

January focused on form elements. I learned about checkboxes, radio buttons, and dropdowns.

Challenge: making forms user-friendly and accessible.

February

February was all about CSS basics. I learned colors, fonts, and background styling.

Challenge: balancing colors and readability.

March

In March, I explored advanced CSS like fonts and text colors. I made the website stylish and unique.

Challenge: creating a visually appealing gangster-style vibe.
